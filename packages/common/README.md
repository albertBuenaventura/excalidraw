# @albertBuenaventura/common

## Install

```bash
npm install @albertBuenaventura/common
```

If you prefer Yarn over npm, use this command to install the Excalidraw utils package:

```bash
yarn add @albertBuenaventura/common
```

With PNPM, similarly install the package with this command:

```bash
pnpm add @albertBuenaventura/common
```
